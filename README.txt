Python Calculator
=================

A simple CLI-based calculator that supports basic arithmetic operations.

Features:
---------
- Addition, subtraction, multiplication, division
- User-friendly console input
- Error handling for invalid input and division by zero

How to Use:
-----------
1. Clone the repository:
   git clone https://github.com/JonSalves/Python-Calculator.git

2. Run the script:
   python calculator.py

3. Follow on-screen instructions to perform calculations.

Requirements:
-------------
- Python 3.x

Future Enhancements:
--------------------
- GUI version with Tkinter or PyQt
- Advanced operations (square root, power, etc.)
- History log of calculations
